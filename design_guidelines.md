# 旅行足迹打卡小程序 - 设计指南

## 1. 品牌定位与设计风格

**应用定位**：旅行足迹打卡记录工具
**目标用户**：旅行爱好者、户外探险者
**设计风格**：手帐风格（温暖、手绘感、文艺）

## 2. 配色方案

采用温暖的手帐风格配色：

```
--primary: #E8B4B8（柔粉 - 主色调，温暖治愈）
--primary-foreground: #FFFFFF
--secondary: #A7C4BC（薄荷绿 - 点亮区域）
--accent: #F4E1D2（米黄色 - 背景底色）
--accent-foreground: #5D4E37（深棕色 - 文字）
--background: #FDF8F3（奶白色 - 页面背景）
--foreground: #5D4E37（深棕色 - 主文字）
--muted: #F5EDE6（浅米色）
--muted-foreground: #8B7355（棕色 - 次要文字）
--border: #E5DDD5（浅棕边框）
--card: #FFFFFF
--card-foreground: #5D4E37
```

**点亮效果**：
- 未点亮区域：填充 `#F0E6DC`，描边 `#C9B99A`
- 已点亮区域：填充 `#A7C4BC`（薄荷绿），描边 `#7FA89B`
- 悬停/点击效果：填充 `#C8DCD4`

## 3. 字体规范

- 标题：20px，粗体，深棕色
- 正文：14px，常规，深棕色
- 标签：12px，中等，棕色

## 4. 间距系统

- 页面边距：16px
- 卡片内边距：16px
- 组件间距：12px
- 列表项间距：8px

## 5. 组件使用原则

- 通用组件（按钮、卡片、标签、弹窗）优先使用 `@/components/ui/*`
- 页面布局使用 View/Text + Tailwind
- 地图组件使用 SVG 实现

## 6. 导航结构

**TabBar 页面**：
1. 首页（地图打卡）
2. 足迹统计

**图标风格**：简约线性图标，与手帐风格一致

## 7. 中国地图实现

- 使用 SVG path 实现各省区域
- 34 个省级行政区（23省+5自治区+4直辖市+2特别行政区）
- 点击区域触发打卡/取消打卡
- 打卡动画：区域颜色渐变 + 星星闪烁效果

## 8. 手帐装饰元素

- 页面顶部：手绘风格标题装饰
- 打卡成功：弹出贴纸/印章效果
- 统计页面：手帐风格图表和卡片
